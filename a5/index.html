Introdução à CSS

CSS é uma linguagem de estilização usada para formatar elementos de um arquivo html. Com ela podemos definir cores, altura, largura, entre outras características, para deixar o html apresentável da forma que quisermos. Veja abaixo alguns exemplos de formatação, utilizando a tag BUTTON como exemplo.

Botão com definições de fonte

A fonte pode ser alterada de diversas formas com CSS. Aqui estão três propriedades como exemplo:

  • font-size: Altera o tamanho da fonte;
  • font-style: Altera o estilo da fonte;
  • color: Altera a cor da fonte.

Botão com definições de borda

As bordas também podem ser alteradas com CSS. Algumas propriedades de exemplo:

  • border: Altera características da borda. Nesse exemplo foi usado "border:none" para tirar a borda;
  • border-color: Altera a cor da borda;
  • border-radius: Altera o raio da borda. Usada para criar bordas arredondadas.
